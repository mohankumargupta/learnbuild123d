# UV and jupyterlab

```sh

python -m ipykernel install --user --name=myenv --display-name "Python (myenv)"  

uv run poe 

jupyter labextension enable jupyterlab-jupytext

uv run --with jupyter jupyter lab
```

need to install extension in jupyterlab for jupytext
